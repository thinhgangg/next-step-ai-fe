
# NextStep Frontend

Frontend cho dự án NextStep, được xây dựng bằng React, TypeScript và Vite.

## Yêu cầu

- Node.js
- npm

## Cài đặt và chạy

```bash
npm install
npm run dev
```

Ứng dụng mặc định chạy tại `http://localhost:5173`.

## Biến môi trường

Tạo file `.env` trong thư mục gốc:

```env
VITE_GRAPHQL_URL=http://localhost:3003/graphql
VITE_GOOGLE_AUTH_URL=
```

## Các lệnh khác

```bash
npm run build
npm run lint
npm run preview
```
